# Projeto integrador - Feira Raízes

## Tecnologias utilizadas
- HTML 5
- CSS + Bootstrap framework

## Proposta do projeto
Desenvolver uma feira digital com foco em divulgar produtores, produtos sazonais e horários de uma feira agroecológica. Com informações de relevantes sobre o produtor e suas respectivas mercadorias.
